# MAX Data Exporter

Расширение для Chrome, которое парсит веб-версию мессенджера МАХ и выгружает сообщения за выбранную пользователем дату в файл CSV или JSON.

## Установка (Добавление в браузер)

> Это расширение пока не опубликовано в Chrome Web Store, поэтому устанавливается вручную из исходников.

### Шаг 1. Скачайте архив с кодом

По [прямой ссылке](https://github.com/Angel-Relations-Group/max_exporter/releases/latest/download/max_exporter.zip)

Или из репозитория:
1. Нажмите кнопку **Code**.
2. Выберите **Download ZIP**.


### Шаг 2. Распакуйте архив

1. Найдите скачанный ZIP-файл.
2. Нажмите правой кнопкой мыши и выберите **Извлечь все** или **Распаковать**.
3. После распаковки у вас появится обычная папка с файлами расширения.
4. Убедитесь, что в распакованной паке находятся файлы расширения, а не ещё одна папка.

### Шаг 3. Откройте страницу расширений Chrome

1. Откройте Google Chrome.
2. В адресной строке введите:

```text
chrome://extensions
```

3. Нажмите Enter.

### Шаг 4. Включите режим разработчика

1. В правом верхнем углу страницы включите **Режим разработчика**.
2. После этого появятся дополнительные кнопки.

### Шаг 5. Загрузите расширение

1. Нажмите кнопку **Загрузить распакованное расширение** / **Load unpacked**.
2. Выберите папку, в которую вы распаковали архив.
3. Подтвердите выбор.
4. После установки расширение появится в списке и рядом с адресной строкой.

## Возможности

- Собирает сообщения за указанную дату.
- Экспортирует результат в CSV-файл или JSON-файл.
- Работает локально в браузере пользователя.

## Как это работает

1. Пользователь открывает веб-версию МАХ.
2. Расширение анализирует сообщения на странице.
3. Пользователь указывает нужную дату.
4. Расширение формирует CSV-файл или JSON-файл и предлагает скачать его.

## Требования

- Google Chrome актуальной версии.
- Включенный режим разработчика для установки расширения вручную.
- Доступ к веб-версии мессенджера МАХ.

## Как пользоваться

1. Откройте веб-версию мессенджера МАХ.
2. Пролистайте до нужной даты или на нужное число страниц / сообщений.
3. Нажмите на иконку расширения.
4. Укажите диапазон дат или нужное число сообщений со страницы.
> При выборе только начальной даты будут выгружены сообщения от этой даты по настоящее время. При выборе только конечной даты сообщения будут выгружены до этой даты включительно. При выборе даты выбранное число сообщений игнорируется и быдут выгружены все сообщения за выбранный период.
5. Запустите экспорт.
6. Сохраните сгенерированный файл.

## Структура файла отчета

Файл отчета содержит сообщения за выбранную дату в табличном виде (CSV) или в виде структуры JSON.

- дата и время сообщения,
- текст сообщения,
- ссылки
- количество просмотров
- количество реакций (общее)

## Обновление расширения

Если вы скачали новую версию с GitHub:

1. Скачайте свежий ZIP-архив.
2. Распакуйте его в ту же папку с заменой файлов.
3. Откройте `chrome://extensions`.
4. Найдите расширение.
5. Нажмите **Обновить** или удалите старую версию и загрузите новую через кнопку **Загрузить распакованное расширение** / **Load unpacked**.

## Безопасность и приватность

- Расширение работает локально в браузере пользователя.
- Расширение не отправляет никакие данные на внешние серверы.

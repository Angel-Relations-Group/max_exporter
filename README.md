# MAX Channel Exporter

**MAX Channel Exporter** — расширение для Chrome и Firefox, позволяющее экспортировать сообщения из веб-версии мессенджера **MAX** в формате **CSV** или **JSON** за выбранный период.

Все данные обрабатываются **локально в браузере** и не передаются на внешние серверы.

---

## Возможности

- 📅 Экспорт сообщений за одну дату или диапазон дат
- 📄 Экспорт в CSV и JSON
- 📦 Разбиение больших выгрузок на несколько файлов
- 🌐 Поддержка Chrome, Edge, Яндекс.Браузера, Brave, Opera и Firefox
- 🔒 Полностью локальная обработка данных
- ⚡ Не требует авторизации в сторонних сервисах

---

## Конфиденциальность

Расширение:

- не отправляет данные на внешние серверы;
- не использует сторонние API;
- работает полностью локально в браузере пользователя.

### Chrome / Chromium

1. Распакуйте архив
2. Откройте:

```
chrome://extensions
```

3. Включите **Режим разработчика**
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку с распакованным расширением

---

### Firefox

1. Распакуйте архив
2. Откройте:

```
about:debugging#/runtime/this-firefox
```

3. Нажмите **Загрузить временное дополнение...** (**Load Temporary Add-on...**)
4. Выберите файл `manifest.json`

> Временное дополнение работает до перезапуска Firefox.

</details>

# Использование

1. Откройте веб-версию MAX.
2. Перейдите в нужный канал.
3. Нажмите на значок расширения.
4. Выберите диапазон дат **или** количество загруженных экранов сообщений.
5. Выберите формат экспорта (CSV или JSON).
6. Запустите экспорт.
7. Сохраните полученный файл.

> Если указана только начальная дата — будут экспортированы сообщения от неё до текущего момента.
>
> Если указана только конечная дата — будут экспортированы сообщения от первого загруженного сообщения до выбранной даты.
>
> При выборе диапазона дат параметр количества экранов игнорируется.

# Что экспортируется

В файл попадают следующие данные:

| Поле | Описание |
|------|----------|
| Дата и время | Время публикации сообщения |
| Ссылка | Ссылка на сообщение |
| Текст | Текст сообщения |
| Просмотры | Количество просмотров |
| Реакции | Общее количество реакций |

Экспорт возможен в формате **CSV** или **JSON**.

# Сборка из исходников

Требуется установленный Node.js.

```bash
node tools/build.mjs
```

После выполнения команды будут созданы директории:

```
dist/chrome
dist/firefox
```

Их можно загрузить в браузер как распакованные расширения.

При публикации релиза GitHub Actions автоматически собирает архивы:

- `max_exporter_chrome.zip`
- `max_exporter_firefox.zip`

### Версия

Версия расширения хранится в поле `version` файла `manifest.json`.
